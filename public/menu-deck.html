The big idea

One edit.
Updated everywhere.

The menu lives in one place — a live database. Every surface that shows the menu reads from that database. Change a price, add a dish, mark something unavailable — and it updates instantly on the digital menu, the screens, the print templates, and the next poster you make.

There is no "menu file". There is no emailing a PDF to the printers. There is no remembering to update the chalkboard and the website and the Instagram.

How to edit the menu

Go to Menu Editor.
Click anything to edit it.

1

Open the admin panel

go.theartyst.co.uk/admin → sign in → Configure → Menu Editor

2

Find the item

Use the category nav at the top to jump to the right section. Categories are numbered so you can see the full menu structure at a glance.

3

Click to edit inline

Click any name, description or price — it becomes editable on the spot. No separate form, no save button hunting. Press Enter or click away to save.

4

Toggle availability

The On toggle marks an item as available on the eat-in menu. The T/A toggle marks it as available for takeaway. Changes are live immediately.

5

Add or hide a category

Use + Add Item within a category, or + New Menu Category for a whole new section. Use Hide Category to remove it from the menu without deleting it.

Smart filtering

Print the right menu
for the right time.

The Print Menu tool knows about your time slots — Brunch, Lunch, Afternoon Tea, Evening. Select one and it filters out items that aren't available at that time, based on the slots you've assigned to each item in the Menu Editor.

Print a brunch menu at 10am. Print an evening bar card at 5pm. The data is already there — the system just filters it for you.

If something looks wrong on the digital menu: hard refresh the page (Ctrl+Shift+R / Cmd+Shift+R). If it still looks wrong, check the item is toggled On in Menu Editor.
